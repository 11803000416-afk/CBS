## 🎯 Supported Image Formats

| Format | Extension | Best For | Mobile Support |
|--------|-----------|----------|-----------------|
| **JPEG** | .jpg, .jpeg | Photos, general | ✅ Universal |
| **PNG** | .png | Quality, transparent | ✅ Universal |
| **WebP** | .webp | Modern, compressed | ✅ Modern devices |
| **GIF** | .gif | Animations, simple | ✅ Universal |
| **HEIC** | .heic | iPhone photos | ✅ Modern devices |
| **TIFF** | .tif, .tiff | Professional | ✅ Some devices |
| **SVG** | .svg | Diagrams, logos | ✅ Modern |
| **BMP** | .bmp | Simple images | ⚠️ Less common |

### Recommended
- **Best:** JPG or HEIC (good quality, small size)
- **Alternative:** PNG (lossless, larger)
- **Modern:** WebP (smallest, best quality)

### Tips for Good Photos
1. **Use good lighting** - Natural light is best
2. **Clear photos** - No blur or motion
3. **Multiple angles** - Front, side, back, interior
4. **Relevant content** - Vehicle condition matters
5. **Clean vehicle** - Looks professional

## 🔧 Troubleshooting

### Issue: Image Won't Upload

**Solutions:**
1. ✅ Check file size: Max 2MB per image
2. ✅ Check format: Use JPG, PNG, or WebP
3. ✅ Check internet: Ensure stable connection
4. ✅ Try again: Browser may have temporary issue
5. ✅ Use different browser: Clear cache if needed

### Issue: Upload Too Slow

**Solutions:**
1. ✅ Check internet speed: Faster = better
2. ✅ Compress image: Use online tool
3. ✅ Use JPG instead of PNG: Smaller size
4. ✅ Upload one at a time: Instead of batch
5. ✅ Try different browser: May have optimizations

### Issue: Can't See Uploaded Images

**Solutions:**
1. ✅ Refresh page: May need to reload
2. ✅ Clear cache: Browser cookies/cache
3. ✅ Wait a moment: Server may still processing
4. ✅ Try different browser: Compatibility issue
5. ✅ Contact admin: If still not visible

### Issue: Image Quality Looks Bad

**Solutions:**
1. ✅ Use better original: Higher quality source
2. ✅ Resize before upload: Make dimensions reasonable
3. ✅ Use good lighting: Better photos = better look
4. ✅ Use WebP format: Better compression
5. ✅ Edit before uploading: Crop, enhance, etc.

## 💡 Pro Tips

### For Sellers
1. **Upload 5-8 photos**
   - Multiple angles = more buyer interest
   - Shows condition better
   - Increases confidence

2. **Use clear photos**
   - Good lighting
   - Clean/polished vehicle
   - Regular daytime photos

3. **Include key areas**
   - Front/back/sides (exterior)
   - Interior/dashboard (condition)
   - Steering wheel/seats (wear)
   - Engine bay (maintenance)
   - Tires/wheels (condition)

4. **Upload in order**
   - Front → Back → Sides → Interior
   - Logical sequence helps buyers

5. **Use recent photos**
   - Shows current condition
   - Don't use old pictures
   - Buyer trust ↑

### For Buying
1. **Check all photos**
   - Look at every angle
   - Zoom in to check condition
   - Notice missing images = red flag

2. **Ask for more photos**
   - Message seller
   - Request specific angles
   - Details matter before purchase

3. **Look for damage**
   - Dents, scratches
   - Interior wear
   - Maintenance signs
