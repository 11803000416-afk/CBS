## 🆘 TROUBLESHOOTING

### Components not showing?
❌ Check CSS is imported in `app.css`  
✅ Solution: Verify line 10 contains `@import './optimization.css';`

### JavaScript functions not available?
❌ Check JS file is included  
✅ Solution: Verify line 24 in layout has `<script src="optimization.js">`

### Styles not applying?
❌ Browser cache  
✅ Solution: Clear cache or do hard refresh (Ctrl+Shift+R)

❌ Removed in older CSS  
✅ Solution: Not needed with new CSS, automatically handled
