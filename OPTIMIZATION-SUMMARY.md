### Troubleshooting

**Components not showing?**
- Make sure optimization.css is imported
- Check component path is correct
- Verify Blade syntax

**JavaScript utilities not working?**
- Confirm optimization.js is loaded (defer attribute)
- Check browser console for errors
- Verify `typeof debounce === 'function'`

**Styles not applying?**
- Clear browser cache
- Rebuild CSS: `npm run build`
- Check CSS is imported in app.css
